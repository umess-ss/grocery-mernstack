
# Grocery MERN Stack

Full-stack grocery delivery web app built with MongoDB, Express, React, and Node.js.

> This project is based on GreatStack's MERN Grocery Delivery tutorial.

## Tech Stack

- React + Vite + Tailwind CSS
- Node.js + Express
- MongoDB
- JWT authentication

## Running locally

```bash
# backend
cd backend
npm install
npm run dev

# frontend
cd ../client
npm install
npm run dev
